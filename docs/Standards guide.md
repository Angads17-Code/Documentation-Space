# Guide for Writing Standards Notes

## Summary Section
Each note should begin with a short summary.

- Capture the main idea of the standard.  
- Keep it to a few sentences.  
- Focus on what the reader must understand at a glance.

## Main Content
After the summary, write the core explanation of the standard.

- Define what the standard covers.  
- Clarify key points, rules, or expectations.  
- Keep wording direct and simple.

## Links to Use
Include a section listing helpful links.

- Add sources, references, or tools relevant to the standard.  
- Only include links that directly support understanding or application.

## Connections and Additional Understanding
End with a brief section explaining how the standard connects to other ideas.

- Show relationships to other standards.  
- Add any extra context that supports clearer understanding.  
- Keep this concise.
